# OpenMemory Guide - 极点 (Pole)

## Overview
极点 (Pole) - 资源的终极汇聚点。一个专注于展示谷歌扩展和精选资源的 docsify 博客。

## Architecture
- Framework: docsify
- Entry point: index.html (expected)

- `docs/_sidebar.md`: Sidebar navigation.

## Patterns
- Bookmark Cards: Custom HTML `<div class="bookmark-card">` for visual link display.
- Plugin Suite: Search, Copy-code, Pagination pre-configured.
