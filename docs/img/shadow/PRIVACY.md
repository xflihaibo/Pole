# 隐私权政策 (Privacy Policy)

**最近更新日期：2026年1月8日**

「伴影 (Shadow Mate)」非常重视用户的隐私。本政策旨在说明我们如何处理您的数据。

### 1. 数据收集原则
我们坚持**数据本地化**原则。
- **无远程服务器**：本扩展程序不设任何后端服务器，您的所有浏览行为数据均不会上传到互联网。
- **本地存储**：所有数据均使用 `chrome.storage.local` 存储在您的计算机本地。

### 2. 我们记录的数据类型
为了提供“私语总结”功能，我们仅在本地记录以下信息：
- **浏览统计**：访问网站的 URL（仅用于分类）、页面标题、停留时长。
- **交互统计**：本地累计点击次数、滚动距离（像素值）、键入字符总数。
- **关键词提取**：从您访问页面的公开元数据（Title/Meta）中提取的关键词。

### 3. 数据保留与删除
- **24小时过期**：所有详细记录会在 24 小时后通过物理方式自动从本地数据库中删除。
- **一键清除**：用户可以通过卸载插件或在设置中手动清除所有本地存储的数据。

### 4. 权限使用说明
- `storage`: 用于本地保存您的统计数据。
- `alarms`: 用于定时触发“归航时刻”提醒。
- `tabs` & `webNavigation`: 用于计算页面的停留时长。
- `scripting`: 用于在页面上显示总结卡片。

### 5. 第三方披露
我们不会向任何第三方披露、出售或共享您的数据。数据始终处于您的物理控制之下。
